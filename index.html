Syntax reference

* + - / ^ with scalars and named objects (B^4, A^-1, 2A + B), plus det(), inv(), trans(), rref(), eig(). Implicit multiplication works: 2A, A(B+B).

Inline matrices work anywhere: [1 2] is a row, [1; 2] a column, [1 2; 3 4] a 2×2 — ; separates rows.
